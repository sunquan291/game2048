# game2048
## 简介

本工程主要是做为Android编码的学习课程，通过开发一个2048的数字游戏，学习了App开发的一般流程，包括控件的使用和重新定义，各种布局的含义。

代码最近在本机基于新的Android版本又调试一次，确定可用。
